## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/Arieviloo/MeetaApp-Frontend

# Go into the front path
$ cd MeetaApp-Frontend

# Install dependencies
$ yarn

# Run the Front
$ yarn start

```
